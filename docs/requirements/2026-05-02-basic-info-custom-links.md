# 基本信息自定义标签与文字链接

## 1. 背景

当前简历编辑器的“基本信息 - 更多”只支持固定字段，GitHub、个人网站、博客也只能输入单个链接字符串。用户希望更多信息可以继续添加自定义标签，同时固定链接字段在简历预览中按“显示文字 + 实际链接”的超文本方式呈现。

## 2. 目标

1. 基本信息支持添加、编辑、删除多个普通文本自定义标签。
2. GitHub、个人网站、博客支持分别填写显示文字与链接地址。
3. 简历预览中链接字段只显示文字，点击文字跳转到对应链接。
4. 兼容已有本地保存或 JSON 导入中的旧链接字符串。

## 3. 范围

1. 修改前端基本信息数据模型、编辑器与模板共享渲染数据。
2. 更新全部内置简历模板对固定文字链接与自定义标签的展示。
3. 保持现有本地保存、JSON 导入导出能力可用。

## 4. 非目标

1. 不新增测试代码、测试目录、测试脚本或测试夹具。
2. 自定义标签本期只支持普通文本，不支持链接。
3. 不调整后端接口、AI 能力、数据库脚本或模板视觉大改。
