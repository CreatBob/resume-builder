# 实时预览滚动与蓝色模板样式回归修复

最新一次提交 `fix: 对齐工作台布局与存储模式入口` 后，简历编辑工作台出现两类前端回归：

1. 实时预览区域无法继续向下滚动，A4 纸底部内容无法完整查看。
2. 蓝色分栏专业模板与蓝色侧栏职场模板的左侧配色等样式异常。
